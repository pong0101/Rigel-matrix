# Rigel Matrix

Rigel Matrix is a lightweight multi-agent AI operating company for Raspberry Pi.

## Agents
- RM-CEO: coordination
- RM-SYS: system monitoring
- RM-ANL: error analysis
- RM-DEV: code assistance
- RM-MEM: project memory

## Supported hardware
- Raspberry Pi Zero 2 W
- Raspberry Pi 4
- Raspberry Pi 5
- Linux mini PC

## Example tasks
- Analyze Python error
- Check system health
- Review project files
- Suggest dashboard improvements

No ESP32 integration in this version.
